# Contact-Management-System-in-C
Contact Management in C

Operations:

[1] Add a new Contact

[2] List all Contacts

[3] Search for contact

[4] Edit a Contact

[5] Delete a Contact

[0] Exit

Steps to Run:

1. Open the contact-management.c file in an Editor.
2. Compile the file.
3. Run the contact-management.exe.
